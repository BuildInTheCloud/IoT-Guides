# How to setup RDP on Pi

## Setup
1. `ssh into pi`
2. `sudo apt-get update`
3. `sudo apt-get upgrade`
4. `sudo apt-get install xrdp`
5. `sudo raspi-config` and set boot to deshtop

## Connect
RDP to your pi using piName.local
